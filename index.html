Discord.js v14 Bot Tutorial

Setting up a Basic Discord Bot

  1. Install Node.js and npm
  2. Create a new project directory
  3. Initialize a new npm project: npm init -y
  4. Install discord.js: npm install discord.js
  5. Create a new file named bot.js

Basic Bot Code


const { Client, GatewayIntentBits } = require('discord.js');
const client = new Client({
  intents: [
    GatewayIntentBits.Guilds,
    GatewayIntentBits.GuildMessages,
    GatewayIntentBits.MessageContent,
  ]
});

client.on('ready', () => {
  console.log(`Logged in as ${client.user.tag}!`);
});

client.on('messageCreate', (message) => {
  if (message.content === 'ping') {
    message.reply('Pong!');
  }
});

client.login('YOUR_BOT_TOKEN');

Upgrading from v13 to v14

  • Flags now use ThisStyle instead of THIS_STYLE formatting
  • MessageContent intent is now required for accessing message content
  • The 'message' event is now 'messageCreate'
